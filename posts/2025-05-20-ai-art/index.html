Today we explore how an LLM-powered script can create striking images on demand, then commit them straight to this repo so Cloudflare Pages deploys them automatically.

The workflow

  • Prompt sent to the image API (e.g. DALL-E 3).
  • Result saved as cover.jpg inside the post folder.
  • Metadata appended to posts.json.
  • CI push triggers a new Pages deployment.

The whole loop finishes in under a minute—global CDN included.
