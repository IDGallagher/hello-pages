Welcome to the new site!

This is the very first post in our Cloudflare Pages powered blog. The entire site is static—every page is just HTML, CSS and a sprinkle of JavaScript.

From now on, adding a post is as easy as:

  1. Creating a new folder in /posts/.
  2. Adding an index.html file following this template.
  3. Appending an object to posts.json.

Push, wait a few seconds, and the site is live worldwide. ✨
